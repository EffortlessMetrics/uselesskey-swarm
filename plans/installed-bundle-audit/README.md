# Installed Bundle Audit

This plan area owns the installed bundle audit and reviewer handoff lane.

The lane starts after the v0.10.0 external-adoption buildout closed. Its job is
to make generated bundles locally auditable from the installed CLI without
turning installed-user commands into repo-local claim proof or release evidence.
