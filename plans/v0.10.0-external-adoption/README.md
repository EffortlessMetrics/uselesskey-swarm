# v0.10.0 External Adoption

This plan area owns the v0.10.0 buildout lane for external adoption and
installed-user workflows.

The lane starts after v0.9.1 shipped and the first-run UX plus
adoption-confidence lanes closed. Its job is to make `uselesskey` useful from a
clean project and installed CLI, not to prepare or cut a release.

- [closeout.md](closeout.md) - Implemented surface, proof, boundaries, and next release-prep handoff
