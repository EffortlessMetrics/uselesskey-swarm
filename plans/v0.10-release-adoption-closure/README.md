# v0.10 Release Adoption Closure

This plan area owns the v0.10.0 release adoption closure lane.

The previous adoption lanes made the path usable from README, Start Here,
external examples, downstream CI guidance, and installed audit handoff. This
lane turns that proven surface into a release-quality product path: profiles
first, receipts first, facade first for Rust users, CLI first for CI users, and
proof-backed docs for every copyable release-facing snippet.

This is not a broad docs lane and not a release execution lane. Publication,
tags, signing, crates.io pushes, GitHub releases, and the final source release
candidate remain explicit source-repo work.
