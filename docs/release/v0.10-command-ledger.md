# v0.10.0 Command Ledger

This ledger freezes release-facing commands and copyable snippets for the
v0.10.0 adoption-closure lane.

Each command maps to its source document, proof command, failure behavior, and
does-not-prove boundary so CI and reviewers do not branch on unstable prose.

## Versioned-command policy

For this lane:

- Release-candidate snippets may mention `0.10.0` only when labeled as candidate
  and not used as copy-paste install guidance until release publication.
- Public install/dependency snippets remain `0.9.1` until source publication.

## Change-control note

This ledger is intentionally narrow: if a command changes, proof expectations and
failure boundaries in this doc must be updated before claim wording is updated.
