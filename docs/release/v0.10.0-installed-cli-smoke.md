# v0.10.0 Installed CLI Smoke

This record binds the release-candidate installed CLI path to command-backed
proof without publishing v0.10.0.

## User Path Covered

The checkout smoke builds the local `uselesskey` CLI and exercises the installed
user loop from a clean project directory:

```bash
uselesskey doctor --format json
uselesskey bundle --profile oidc --out target/uselesskey-oidc
uselesskey verify-bundle --path target/uselesskey-oidc
uselesskey inspect-bundle --path target/uselesskey-oidc
uselesskey audit-bundle --path target/uselesskey-oidc --ci --expect-profile oidc --policy strict
```

The same bundle, verify, strict CI audit, and inspect path runs for the
`scanner-safe`, `tls`, `oidc`, and `webhook` profiles in
`cargo xtask external-adoption-smoke --path .`.

## Proof Commands

```bash
cargo test -p xtask external_adoption
cargo test -p uselesskey-cli --all-features audit_bundle
cargo xtask external-adoption-smoke --path .
```

## Generated Evidence

The smoke writes reviewable evidence under:

```text
target/external-adoption-smoke/report.json
target/external-adoption-smoke/report.md
target/external-adoption-smoke/logs/
target/external-adoption-smoke/work/
```

The receipt includes:

- `cli-doctor-json`;
- `cli-bundle-<profile>`;
- `cli-verify-<profile>`;
- `cli-audit-<profile>` with `--ci --expect-profile <profile> --policy strict`;
- `cli-inspect-<profile>`.
